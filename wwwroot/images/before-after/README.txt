Drop your before/after photos in this folder using these exact file names
(any one extension per image): 

  before.jpg / before.jpeg / before.png / before.webp
  after.jpg  / after.jpeg  / after.png  / after.webp

The homepage "Cheaper and faster" slider automatically picks them up on
page load -- no code changes needed. Use images with the same aspect
ratio and framing (ideally the same building, before and after cleaning)
for the best slider effect.
