Drop your hero photos in this folder using these exact file names
(any one extension per slide):

  slide-1.jpg / slide-1.jpeg / slide-1.png / slide-1.webp
  slide-2.jpg / slide-2.jpeg / slide-2.png / slide-2.webp
  slide-3.jpg ...
  up to slide-10

The homepage hero slideshow automatically picks them up, in numeric
order, on page load -- no code changes needed. Use widescreen photos
(landscape, similar aspect ratio) of drones in action, cleaned facades,
or finished jobs for the best result.

If no slides are found, a placeholder with upload instructions is shown
in their place instead of a broken slideshow.
